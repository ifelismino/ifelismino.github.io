Featured Project

GPU-Accelerated Python DMRG

Undergraduate Research Topic (December 2018)

A CPU-GPU hybrid approach for Density Matrix Renormalization Group calculations was developed for this project. Matrix-Matrix and Matrix-Vector operations are done in the GPU provided that the scale of the operation is large enough for increase in calculation speed to overtake the data copying overhead.

As can be seen, a GPU-accelerated Jacobi-Davidson algorithm was successful produced. However, compared to the SciPy implementation of an eigensolver - the GPU-accelerated algorithm loses. This can be attributed to two main factors. The first reason is that the two eignsolvers uses different algorithms, and the advantage of using one algorithm vs using the other can depend on the diagonal dominance (r) of the matrix. The second is the difference in overhead of using a singular C function wrapped for Python (SciPy) vs a Python code that calls each operation from Python.

Modified from Leo GiggleLiu's Jacobi-Davidson Algorithm.
Using Python 3.6 Source code
